# Execução do código

Para executar o código, dar os seguintes comandos:

## No Linux
``` PowerShell
gcc -c *.c
gcc -o cvet.out *.o
./cvet.out
```

## No Windows
``` PowerShell
gcc -c *.c
gcc -o cvet.out *.o
cvet.out
```
